# Documentation Structure

This directory contains all project documentation organized by topic and purpose.

## üìÅ Directory Structure

### `/CharacterCustomization/`
Documentation related to character customization features, inventory UI, and character system guides.

### `/DungeonCategoryFix/`
Documentation and guides for the dungeon category matching fix implementation.

### `/Frontend/`
Frontend-specific documentation including setup guides, connection guides, and project structure.

### `/Guides/`
General guides and how-to documents for team members and developers, including testing guides.

### `/ItemEffects/`
Documentation about item effects, rewards system, and item-related features.

### `/Services/`
Backend services documentation organized into subdirectories:

#### `/Services/API/`
- API documentation and endpoints
- Architecture overview
- Authentication setup
- Content endpoints status

#### `/Services/Database/`
- Database seeding guides
- Question population documentation
- Quick reference guides

#### `/Services/Deployment/`
- Deployment guides and procedures

#### `/Services/Implementation/`
- Implementation details for various features
- System implementations (caching, rewards, etc.)
- Debugging guides

#### `/Services/` (root)
- Project structure overview
- Services overview

### `/Summaries/`
Session summaries, final status reports, and project completion summaries.

### `/Archive/`
Archived documentation from previous development phases, sprints, and sessions. Contains historical documentation, fix guides, implementation notes, and testing documentation.
